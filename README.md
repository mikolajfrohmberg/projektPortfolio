Poniższe repozytorium zawiera kod źródłowy wykonanego przeze mnie portfolio.
Obecnie nie jest ono jeszcze dostępne pod adresem www.

- JQuery 3.4.1
- Slick 1.8.1
- MomentJS 2.24
